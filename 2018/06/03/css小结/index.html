清除浮动的几种方式

清除浮动是为了清除使用浮动的元素造成的影响；浮动的元素，高度会塌陷，对布局不利。
解决方法：
1.给父级div设置高度height
2.使父级div也一起浮动
3.给父级元素设置 overf:hidden;
4.使用clearfix类名


div的几种居中方式

水平居中

1.给div设置宽度，并使用margin：0 auto;

2.使用绝对定位

水平垂直居中

1.使用绝对定位

2.使用transform属性

3.使用弹性布局
